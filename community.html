How to Contribute

FOSScraft is a community-driven project. There are many ways to help:

Code Contributions

Help improve FOSScraft by contributing code, fixing bugs, or adding features. Check out our GitHub repository.

Translations

Help translate FOSScraft into your language on Weblate.

Reporting Bugs

Found a bug? Report it on GitHub Issues to help us improve.

Feature Suggestions

Have an idea? Open a discussion or issue on GitHub to share your suggestions.

Documentation

Help improve documentation and wiki pages. Great way to contribute if you're not a developer.

Support & Help

Answer questions and help other community members on Discord, Matrix, or Reddit.

Community Guidelines

FOSScraft is committed to providing a welcoming and inclusive community. Please review our Code of Conduct.

  • Be respectful and inclusive
  • No harassment, discrimination, or hateful content
  • Keep discussions on-topic
  • Respect others' opinions and ideas
  • Report inappropriate behavior to moderators
